# Hackathon 2026 Listing

A modern, responsive website listing major upcoming hackathons in 2026. Built with Next.js and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This project is ready to be deployed on [Vercel](https://vercel.com).

1. Push this code to a GitHub repository.
2. Import the project in Vercel.
3. Vercel will automatically detect Next.js and configure the build settings.
4. Click **Deploy**.

## Features
- **Search & Filter**: Find hackathons by name, organization, or status.
- **Responsive**: Optimized for mobile and desktop.
- **Dark Mode**: Automatically adapts to system preference.
